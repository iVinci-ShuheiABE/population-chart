# population-chart

React、Vite、およびTypeScriptを使用して人口推移グラフを表示します。

## インストール

1. リポジトリをクローン:

   ```bash
   git clone https://github.com/your-username/population-chart.git
   cd population-chart
   ```

2. 依存関係をインストール:

   ```bash
   npm install
   ```

## 使用方法

1. 開発サーバーを起動:

   ```bash
   npm run dev
   ```

2. ブラウザを開き、`http://localhost:5173/` にアクセスし、確認。
